# 康威生命游戏

这是一个用 Qt 写的生命游戏。

生命游戏是剑桥大学约翰·何顿·康威设计的二维元胞自动机模型。该模型在无限的二维网格中运行，每个细胞的状态由周围8个相邻细胞数量决定，每个细胞只有两种状态：活着或死亡。

基本规则：

- 活细胞周围有 2 或 3 个活邻居，会继续存活。
- 死细胞周围刚好有 3 个活邻居，会变成活细胞。
- 其他情况都会变成死亡。


## 数据存储

程序实际模拟的网格/画布是有限大小的。

画布默认是 10000 x 10000，如果用二维数组会浪费很多空间，所以这里只保存活细胞：

```cpp
QSet<QPoint> liveCells;
```

`QPoint` 表示一个细胞的逻辑坐标，比如第几列、第几行。

## 边界处理

程序支持边界环绕。

关闭边界环绕时，超出画布的坐标直接无效。

开启边界环绕时：

- 左边越界会回到右边。
- 右边越界会回到左边。
- 上下边界也是一样。

这部分由 `normalizeCell()` 处理。

## 更新方式

更新函数是 `stepSimulation()`。

更新步骤：

1. 遍历当前所有活细胞。
2. 统计每个活细胞周围 8 个邻居。
3. 用 `QHash<QPoint, int>` 记录每个候选格子的邻居数量。
4. 根据生命游戏规则生成下一代活细胞。
5. 用新集合替换旧集合。
6. 代数加一，刷新画面和统计信息。

只有活细胞附近的格子才可能发生变化。远处没有活细胞的空白区域，不可能突然产生新细胞。所以这里没有遍历整个画布，性能更好。

## 显示画布

绘制流程：

1. 先把整个控件画成深灰色。
2. 再把逻辑画布区域画成黑色。
3. 计算当前窗口能看到哪些逻辑格子。
4. 遍历活细胞集合。
5. 不在可见范围内的细胞跳过。
6. 可见的细胞转换成屏幕坐标。
7. 用白色矩形画出来。

此外还支持缩放与拖动。
